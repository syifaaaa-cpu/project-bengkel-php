# Project Bengkel PHP Native

Project sistem login dan register dengan enkripsi password dan hak akses (role).

## Fitur:
- **Login & Register**: Alur masuk dan daftar akun.
- **Enkripsi Password (Hash)**: Keamanan password menggunakan `password_hash`.
- **Role User**: Pembatasan hak akses untuk Admin dan Pelanggan.

## Cara Penggunaan:
1. Import database `db_bengkel.sql` ke phpMyAdmin.
2. Konfigurasi `koneksi.php`.
3. Buka `login.php` di browser.
